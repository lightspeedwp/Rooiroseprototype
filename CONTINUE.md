# 🚀 Quick Continue Prompt

Copy and paste this to resume work:

```
Continue with the next task from /tasks/CURRENT-TASKS.md
```

---

**That's it!** The AI will:
1. Read `/tasks/CURRENT-TASKS.md`
2. Find the next incomplete task
3. Execute it according to the task description
4. Update progress
5. Confirm completion

---

**Alternative prompts**:

**For status check**:
```
What's the status of /tasks/CURRENT-TASKS.md? What's next?
```

**For specific task**:
```
Do Task 2.1 from /tasks/CURRENT-TASKS.md (Classifieds Page)
```

**For help**:
```
Read /tasks/CONTINUE-PROMPT.md and explain how to resume work
```
