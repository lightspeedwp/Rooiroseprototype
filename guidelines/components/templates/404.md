# 404 Template

**Last updated**: 2026-02-25
**Category**: WordPress Template
**Template file**: `/wordpress-export/themes/die-papier-theme/templates/404.html`
**Pattern file**: `/wordpress-export/themes/die-papier-theme/patterns/page-404.php`
**React source**: `/src/app/pages/NotFound.tsx`

---

## 1. Purpose

WordPress FSE template for the 404 Not Found page. Rendered automatically by WordPress when no content matches the URL. Uses the `die-papier/page-404` pattern for the main content area.

## 2. Template Structure

The `404.html` template follows the standard Die Papier template anatomy:

```
<!-- wp:template-part {"slug":"header"} /-->
<!-- wp:pattern {"slug":"die-papier/page-404"} /-->
<!-- wp:template-part {"slug":"footer"} /-->
```

## 3. Pattern Structure (`die-papier/page-404`)

```
┌─ Group (constrained, 600px, full-align, vertical spacing 80)
│  ├─ H1 "404" (6rem, text-primary, center-aligned)
│  ├─ H2 "Jammer! Bladsy nie gevind nie." (center-aligned)
│  ├─ P "Die bladsy bestaan nie, of is verskuif. Probeer een van die skakels hieronder." (center-aligned)
│  └─ Buttons (flex, center-justified)
│     ├─ "Gaan na tuisblad" (fill style, href="/")
│     ├─ "Soek" (outline style, href="/soek")
│     └─ "Kontak ons" (outline style, href="/kontak")
│
├─ Group (section-muted, constrained 600px)
│  ├─ H3 "Gewilde bladsye" (center-aligned)
│  └─ Navigation (flex, center-justified, gap spacing|30)
│     ├─ Nuus → /nuus
│     ├─ Sport → /sport
│     ├─ E-uitgawes → /e-uitgawes
│     ├─ Kompetisies → /kompetisies
│     └─ Kontak → /kontak
```

## 4. Pattern Registration

```php
/**
 * Title: 404 Bladsy Nie Gevind
 * Slug: die-papier/page-404
 * Categories: page
 * Inserter: false
 */
```

- `Inserter: false` — prevents editors from inserting this pattern in arbitrary locations.

## 5. Block Inventory

| Block | Usage |
|:------|:------|
| `core/group` | 2× — main error section + popular pages section |
| `core/heading` | 3× — H1 "404", H2 title, H3 "Gewilde bladsye" |
| `core/paragraph` | 1× — description text |
| `core/buttons` | 1× — 3 action buttons |
| `core/button` | 3× — home (fill), search (outline), contact (outline) |
| `core/navigation` | 1× — popular page links |
| `core/navigation-link` | 5× — Nuus, Sport, E-uitgawes, Kompetisies, Kontak |

## 6. Design Tokens Used

- **Colors**: `color|primary` (404 heading)
- **Typography**: 6rem (404 heading), default preset sizes for H2/H3/P
- **Section style**: `is-style-section-muted` on popular pages group

## 7. React ↔ WordPress Parity

| Feature | React (`NotFound.tsx`) | WordPress (`page-404.php`) |
|:--------|:----------------------|:--------------------------|
| 404 code | H1, text-8xl, font-heading | H1, 6rem, text-primary |
| Title | H2, text-3xl | H2, default |
| Description | P, text-lg | P, default |
| Buttons | 3× Button component | 3× core/button |
| Popular pages | Grid cards (2→3 col) | Navigation links (flex) |
| Help section | ❌ Removed | ❌ Not included |
| Back button | ❌ Removed | ❌ Not included |

## 8. Dependencies

- Template parts: `header`, `footer`
- Section styles: `is-style-section-muted`

## 9. Known Exemptions

None.
