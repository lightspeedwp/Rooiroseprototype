# Template: page.html

**Last updated**: 2026-02-23
**Category**: Template
**WP file**: `/wordpress-export/themes/die-papier-theme/templates/page.html`
**React equivalent**: Static info pages, policy pages, etc.

---

## 1. Purpose

Default page template for static WordPress pages. Includes breadcrumbs and post content within a constrained layout.

## 2. Structure

```
header (template part)
└─ main (constrained, 1280px)
   ├─ breadcrumbs (template part)
   └─ post-content (constrained)
footer (template part)
```

## 3. Template Parts Used

- `header`
- `breadcrumbs`
- `footer`

## 4. Layout

- Type: `constrained`
- Content size: `1280px`

## 5. Used By React Pages

About, Contact, Team, FAQ, Advertise, Sitemap, and most static pages.
