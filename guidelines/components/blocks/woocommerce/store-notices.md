# WooCommerce Block: Store Notices

**Last updated**: 2026-03-03
**Block Name**: `woocommerce/store-notices`
**Category**: `woocommerce`
**Display Name**: Store Notices

---

## Overview

Display shopper-facing notifications generated by WooCommerce or extensions. Includes success messages, error alerts, and informational notices.

---

## Block Specification

- **Name:** `woocommerce/store-notices`
- **Supports:** align (full, wide), interactivity (clientNavigation), multiple
- **Attributes:** `align`

---

## Block Styles

No custom block styles. Uses WooCommerce default styling, overridden by theme CSS.

---

## Pattern Usage

Used in **5 commerce patterns** for consistent notification display:

| Pattern | File |
|---------|------|
| `die-papier/woo-cart` | `woo-cart.php` |
| `die-papier/woo-checkout` | `woo-checkout.php` |
| `die-papier/woo-my-account` | `woo-my-account.php` |
| `die-papier/woo-order-confirmation` | `woo-order-confirmation.php` |
| `die-papier/woo-single-product` | `woo-single-product.php` |

**Placement**: Always at the top of the commerce content area, before the main block.
