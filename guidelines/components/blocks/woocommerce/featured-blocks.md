# WooCommerce Blocks: Featured Product & Category

**Last updated**: 2026-03-03
**Block Family**: `woocommerce/featured-product`, `woocommerce/featured-category`
**Category**: `woocommerce`

---

## Overview

Hero-style blocks for visually highlighting a specific product or category with a cover image, overlay, and CTA button.

---

## Block Inventory

| Block | Description | Key Supports |
|-------|-------------|-------------|
| `featured-product` | Highlight a product/variation | align, ariaLabel, color, filter (duotone), spacing (padding) |
| `featured-category` | Highlight a product category | align, ariaLabel, color, filter (duotone), spacing (padding) |

### Shared Attributes
Both blocks share: `alt`, `contentAlign`, `dimRatio`, `focalPoint`, `hasParallax`, `imageFit`, `isRepeated`, `linkText`, `mediaId`, `mediaSrc`, `minHeight`, `overlayColor`, `overlayGradient`

### Featured Product Additional Attributes
`previewProduct`, `productId`

### Featured Category Additional Attributes
`categoryId`, `previewCategory`

---

## Block Styles

No custom block styles.

---

## Pattern Usage

Not currently used in Die Papier patterns. E-edition promotion uses custom hero patterns with `dp/slider` instead.
