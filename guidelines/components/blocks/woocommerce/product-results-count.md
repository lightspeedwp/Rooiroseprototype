# WooCommerce Block: Product Results Count

**Last updated**: 2026-03-03
**Block Name**: `woocommerce/product-results-count`
**Category**: `woocommerce`
**Display Name**: Product Results Count

---

## Overview

Display the number of products on the archive page or search result page (e.g. "Wys 1–12 van 24 resultate").

---

## Block Specification

- **Name:** `woocommerce/product-results-count`
- **Supports:** color (text, background), interactivity (clientNavigation), typography (fontSize)
- **Attributes:** (none)

---

## Block Styles

### Default (`wc-product-results-count-default`)
**File**: `styles/blocks/woocommerce/product-results-count/default.json`

---

## Pattern Usage

Available for product archive templates. Not currently used in custom Die Papier patterns.
