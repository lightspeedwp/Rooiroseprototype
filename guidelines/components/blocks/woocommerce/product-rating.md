# WooCommerce Blocks: Product Rating

**Last updated**: 2026-03-03
**Block Family**: `woocommerce/product-rating`, `product-rating-counter`, `product-rating-stars`, `product-average-rating`
**Category**: `woocommerce-product-elements`

---

## Overview

A family of blocks for displaying product ratings. Die Papier does not currently use product ratings for e-editions, but the blocks are styled for future use.

---

## Block Inventory

| Block | Description | Ancestor | Supports |
|-------|-------------|----------|----------|
| `product-rating` | Average rating display | all-products, single-product, product-template, post-template | color, spacing, typography (fontSize) |
| `product-rating-counter` | Review count (e.g. "5 resensies") | single-product | color, spacing, typography (fontSize) |
| `product-rating-stars` | Star rating visual | single-product | color, spacing, typography (fontSize) |
| `product-average-rating` | Average rating number | single-product | color, spacing, typography (fontSize) |

---

## Block Styles

### Default (`wc-product-rating-default`)
**File**: `styles/blocks/woocommerce/product-rating/default.json`

---

## Pattern Usage

Not currently used in Die Papier patterns. E-edition products don't have customer reviews enabled.
