## Common Structure

```
┌─ <div> (bg-white dark:bg-background, min-h-screen)
│  ├─ PageContainer (breadcrumbs: [Beleid, page-name])
│  ├─ ContentHero (title, subtitle, image from HERO_IMAGES.legal)
│  └─ .alignwide py-10
│     └─ .prose .prose-lg .max-w-none (text-gray-700 dark:text-gray-300)
│        └─ Long-form Afrikaans legal content (hardcoded in each file)
```

## WordPress Mapping

### Construct Type
Static pages — content entered directly in the WordPress block editor using `core/heading`, `core/paragraph`, `core/list`, and `core/group` blocks.

### Implementation Notes
1. Create each page in WP with the appropriate title
2. Set parent page to "Beleid" for all child policy pages
3. Use the `page.html` or `page-no-title.html` template
4. Content hero via `core/cover` block (the `dp/content-hero` block is deprecated — use `core/cover` with overlay colour preset)
5. Breadcrumbs auto-generated by Yoast SEO or the breadcrumbs template part

### Existing WP Files
- Template: `/wordpress-export/themes/die-papier-theme/templates/page.html`
- Template: `/wordpress-export/themes/die-papier-theme/templates/page-no-title.html`
