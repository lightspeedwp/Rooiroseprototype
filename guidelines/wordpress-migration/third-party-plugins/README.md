# Third-Party Plugin Integration

**Last updated**: 2026-03-01

This directory contains integration guides for all third-party WordPress plugins used in the Die Papier block theme.

## Files

| File | Plugin | Purpose |
|---|---|---|
| `advanced-ads.md` | Advanced Ads (Pro) | Ad placements, sticky ads, display conditions, membership ad hiding |
| `advanced-query-loop.md` | Advanced Query Loop | Taxonomy queries, article deduplication, homepage section configs, caching |
| `gravity-forms.md` | Gravity Forms | Form builder, newsletter signup, contact forms, Afrikaans translations |
| `icon-block.md` | Outermost Icon Block | Icon library, custom SVG icons, size mapping |
| `plugin-structure.md` | Die Papier Blocks (custom) | CPTs, custom blocks, taxonomies, SCF fields |
| `social-sharing.md` | Outermost Social Sharing | Facebook, WhatsApp, X, Email, Copy Link sharing |
| `yoast-seo.md` | Yoast SEO | Breadcrumbs, FAQ block, Schema.org structured data |

## Naming Convention

Files are named after the plugin slug without the `-integration` suffix. The `third-party-plugins/` directory name already implies integration context.
