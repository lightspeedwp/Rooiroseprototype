# Inc File: custom-icons.php

**File**: `inc/custom-icons.php`  
**GitHub**: https://github.com/lightspeedwp/die-papier-tema/blob/develop/inc/custom-icons.php  
**Purpose**: Register custom SVG icons for the Outermost Icon Block plugin

---

## Responsibilities

- Register Die Papier custom icon collection with the Outermost Icon Block
- Provide brand-specific icons not available in the default icon library
- Icons are registered via the `outermost_icon_block_collections` filter
