## 🎨 **DESIGN SYSTEM HIGHLIGHTS**

### **Typography**
- **Headings**: Playfair Display SC (small caps serif)
- **Body**: Karla (clean, modern sans-serif)
- **Sizes**: Fluid scale from text-xs (12px) to text-5xl (48px)
- **Custom Variables**: `--fvs-h1`, `--lh-h1`, `--ls-h1` for precise control

### **Colors**
- **Primary Red**: #e01e12 (brand-red) — CTAs, accents, active states
- **Navy**: #172134 (brand-navy) — Headings, primary text
- **Grays**: 50-900 scale for backgrounds, text, borders
- **Dark Mode**: Full color inversion with proper contrast ratios

### **Spacing**
- **Base Unit**: 8px (Tailwind default)
- **Gaps**: gap-4 (16px), gap-6 (24px), gap-8 (32px)
- **Padding**: p-6, p-8, p-10 for cards and sections
- **Margins**: mb-6, mb-8, mb-10, mb-12 for vertical rhythm

### **Components**
- **Cards**: rounded-xl, shadow-sm, hover:shadow-md
- **Buttons**: h-12, px-6, rounded-lg, font-bold
- **Pills**: rounded-full, px-4, py-2, text-sm
- **Numbered Circles**: 40px, brand-red, white text (for steps)
- **Icons**: 16-24px (UI), 56-64px (hero/confirmation)

## 🚀 **TECHNICAL STACK**

### **Frontend**
- **Framework**: React 18 + TypeScript
- **Routing**: React Router v7 (Data mode)
- **Styling**: Tailwind CSS v4 + Custom CSS
- **Icons**: Lucide React
- **Charts**: Recharts (lazy-loaded)
- **Carousels**: React Slick (lazy-loaded)
- **Build Tool**: Vite 6
- **State Management**: React Context + Hooks

### **Features**
- **Dark Mode**: System preference + manual toggle (localStorage)
- **Lazy Loading**: Route-based code splitting
- **Responsive**: Mobile-first, 3 breakpoints
- **Accessibility**: WCAG 2.1 AA compliant
- **SEO**: Meta tags, Open Graph, structured data
- **Performance**: Lighthouse score > 90 target
