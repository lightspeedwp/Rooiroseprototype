# Local Font Assets

To ensure GDPR compliance and optimal performance, this theme uses locally hosted font files instead of Google Fonts CDN.

## Required Font Files

Please download the following variable font files from Google Fonts (or your preferred source) and place them in this directory:

1.  **Inter (Variable)**
    *   **Filename:** `Inter-VariableFont_slnt,wght.ttf`
    *   **Source:** [Google Fonts - Inter](https://fonts.google.com/specimen/Inter)
    *   **Usage:** Used for body text, UI elements, and headings.

2.  **Roboto Serif (Variable)**
    *   **Filename:** `RobotoSerif-VariableFont_GRAD,opsz,wdth,wght.ttf`
    *   **Source:** [Google Fonts - Roboto Serif](https://fonts.google.com/specimen/Roboto+Serif)
    *   **Usage:** Used for large headings and display text. Supports variable axes for Optical Size, Width, Weight, and Grade.

## Configuration

These fonts are registered in `theme.json` under `settings.typography.fontFamilies`. If you change the filenames, please update `theme.json` accordingly.
